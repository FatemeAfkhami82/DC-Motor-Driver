# 24V DC Motor Driver Project
This repository contains the design files, source code, and documentation for a professional 24V DC motor driver.

## 🛠 Project Overview
- **Core Functionality:** Designed for precise motor control using STM32/Microcontroller.
- **PCB Design:** Developed using Altium Designer, featuring multi-layer layout and power optimization.
- **Firmware:** Firmware developed in C for efficient hardware-software interaction.

## 📁 Project Structure
- **/PCB_ProjectE1:** Main Altium Designer project files.
- **/codes:** Source code (C) for motor control algorithms.
- **/Components (قطعات):** BOM and hardware component specifications.
- **/Report (گزارش کار):** Detailed project analysis and testing documentation.
